# Best Practices for Using Ross XPression Graphite

1. **Keep Templates Organized:** Use logical folder structure and naming conventions.
2. **Test Everything Before Live:** Always simulate animations and transitions.
3. **Optimize Graphics:** Reduce layer complexity for smoother playback.
4. **Document Changes:** Maintain version history for templates and assets.
5. **Leverage Hotkeys & Shortcuts:** Save time during live operations.
6. **Regular Backups:** Avoid losing work with frequent saves and backups.
